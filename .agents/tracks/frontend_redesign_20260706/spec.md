# Track Specification: Frontend Redesign

## Overview
Complete reconfiguration and redesign of the VendorMind frontend dashboard, landing page, and onboarding pages using the new green polygonal V logos.

## Functional Requirements
- **FR-1**: Copy and rename `1.png` and `2.png` to Vite asset public directory.
- **FR-2**: Dashboard sidebar displays new logo-dark asset instead of generic cart icon.
- **FR-3**: Marketing landing page uses a premium space-blue background, styled chat mockups, and pricing cards matching the inspo blueprints.
- **FR-4**: Onboarding wizard layout utilizes a connected vertical timeline progress bar on the left with circular steps and helper cards.
- **FR-5**: Conversations chat inbox adopts a 3-column dark layout with collapsible list groups and take-over reply feeds.
- **FR-6**: Catalog manager displays outline statistics cards and tables with inline thumbnails.

## Acceptance Criteria
- [ ] Frontend compiles successfully using `bun run build`.
- [ ] Main sidebar, landing, onboard, conversations, products, and wallet pages render all content correctly.
- [ ] WhatsApp message processing and checkout smoke tests pass successfully.
